# DevWeb Desafio 1

<h2> Objetivo: </h2>
<p>Criar um site de uma universidade fictícia chamada UNES, com as tecnologias de HTML e CSS, com 3 telas, sendo elas: home, quem somos e contato </p>

<h3>Tecnologias utilizadas: </h3>
<ul>
  <li> HTML5 </li>
  <li> CSS3 </li>
</ul>
